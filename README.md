ZYBO_LED_test
=============

Digilent社のZYBO用のLEDテスト用プロジェクトです。ISE14.7を使用しています。
LEDをインクリメントするAXI Lite SlaveのカスタムIPをXPSプロジェクトに追加してあります。
プロジェクトを作製した経緯については、以下のブログをご覧ください。
